# ☕ Coffee Shop Sales Analysis Dashboard (Microsoft Excel)

## Project Overview

This project analyzes Coffee Shop Sales data using Microsoft Excel.

The main goal of this project is to clean the raw data, transform it into a structured format, analyze sales performance, and build an interactive dashboard that helps understand business performance and customer buying patterns.

---

## Tools Used

- Microsoft Excel
- Power Query
- Power Pivot
- Pivot Tables
- Pivot Charts
- Slicers

---

## Data Cleaning & Transformation

The following data cleaning and transformation tasks were performed using Power Query:

- Imported the dataset into Excel using Power Query.
- Removed unwanted text values (**LG, RG, SM**) from the Product Detail column.
- Removed extra spaces using the **Trim** function.
- Created a new **Size** column (Small, Regular, Large, Not Defined).
- Created a **Total Bill** column by multiplying **Quantity × Unit Price**.
- Cleaned and formatted the Time column.
- Changed data types where required.
- Extracted:
  - Month Name
  - Month Number
  - Day Name
  - Day Number
  - Hour

## 📊 Data Analysis

This project answers the following business questions:

- What is the total sales revenue?
- How do sales vary by month?
- How do sales vary by day of the week?
- What are the peak sales hours?
- Which store location generates the highest sales?
- Which store has the highest customer footfall?
- Which product categories perform the best?
- Which products generate the highest revenue?
- What is the total quantity sold?
