# Coffee-Shop-Sales
This project analyzes Coffee Shop Sales data using Microsoft Excel.

The goal of this project is to clean the data, analyze sales performance, and build an interactive dashboard that helps understand business performance.

Tools Used
Microsoft Excel
Power Query
Power Pivot
Pivot Tables
Pivot Charts
Slicers
Data Cleaning

In this project, I performed the following data cleaning tasks:

Imported data using Power Query
Removed unwanted text values (LG, RG, SM)
Removed extra spaces using Trim
Created a new Size column
Created Total Bill column using Quantity × Unit Price
Cleaned the Time column
Changed data types
Extracted:
Month Name
Month Number
Day Name
Day Number
Hour
